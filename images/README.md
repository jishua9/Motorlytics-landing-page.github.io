# Screenshot assets

Drop real app screenshots here using these exact filenames — the pages pick them
up automatically (no markup changes). Until a file exists, a labelled placeholder
box shows in its place.

| Filename | Where | Capture | Required |
|----------|-------|---------|----------|
| `hero-telemetry.png` | Home hero panel | Telemetry track map (speed-coloured GPS trace + corner markers + data overlay) | Yes |
| `feature-telemetry.png` | Home deep-dive 1 | Charts tab (multi-channel + lap-delta) or corner analysis | Yes |
| `feature-pitwall.png` | Home deep-dive 2 | Pit Wall (countdown + weather + rain radar + schedule) | Yes |
| `feature-car-costs.png` | Home deep-dive 3 | Car profile (parts lifecycle + services) and/or spending summary | Yes |
| `dashboard.png` | (optional) | Driver dashboard at-a-glance | No |
| `results.png` | (optional) | Results — position + lap-time charts | No |

Capture in **dark theme**, widescreen (~16:10), 2×/retina, real data, no PII.
